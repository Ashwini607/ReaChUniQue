Retrieve information from the ChEMBL, UniProt and Reactome triple stores using the semantic web technology.

This tool has a form which allows to select the options according to your query related to compounds and their proteins target and then submit. After submission, it generates the background SPARQL query that runs on the ChEMBL SPARQL endpoint and provides the results along with the generated SPARQL query.
