Retrieve information from ChEMBL, UniProt and Reactome triple stores using semantic web technology.

This tool has a form to select the options according to your problem related to compounds targeting proteins and then submit. After submission, it generates the background SPARQL query that runs on ChEMBL SPARQL endpoint and provides the results along with the generated SPARQL query.
