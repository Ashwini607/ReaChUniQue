Connectivity

Here you can see the connectivity among options present in the form. This tells about schema for this application. The similar schema image exists for triple store. The tool has followed triple store schema. Here have tried to represent in a bit more simple way.

You can see how the first box options are interconnected to each others. so if you will be interested in getting information regarding substance and their target then you would have to select the Activity and Assay option too, to make a link between Substance and their Target. Apart from the first box, other boxes elements are independent of each other.

What does the term written inside the circular node mean, has been described in the Documentation page.
