Connectivity

Here you can see the connectivity among options present in the form. This is about schema for this application. Similar schema image exist for triple store. Tool has followed triple store schema and tried to represent in bit simple way that could help in understanding the actual schema image of triple store.

You can see how the first box options are interconnected to each others. so if you will be interested in getting information regarding Substance and their Target then you would have to select the Activity and Assay option too to make link between Substance and their Target. Apart from first box, other boxes elements are independent of each other.

If you want to filter elements by writing in box then make sure you have selected the corresponding option too. For example, You are interested in targeting Homo Sapiens organism then beside writing organism name in box below Organism option you would have to select the Organism option too.
